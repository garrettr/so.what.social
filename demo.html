So.What.Social JQuery Plugin

so.what.social is a social media aggregator. Given a list of feeds, it grabs the most recent activity and displays a customizable feed to the user.

Everything is done with YQL, service-provided RESTful API's, and JSONP - so there is no need for any server-side code to retrieve your recent social network activity.

so.what.social is a work in progress, but is stable and easy to use. At the moment we support Twitter, Facebook, and any RSS/Atom feed. Head over to my github to get the code, and check out the README for simple setup instructions.

so.what.social is based on John Patrick Given's excellent So So Social JQuery plugin. so.what.social is so.named because I started writing it while listening to Kind of Bloop.
